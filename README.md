# 🚀 Rayvern

A simple and efficient tool for batch file renaming and text replacement.

Developed through the collaboration of the **Wyvern** and **Rayka** teams.

## ✨ Features

* Batch process multiple files at once
* Replace only the specified text segment
* Preserve all other file content unchanged
* Fast and lightweight
* Open-source and transparent

## 🎯 Use Cases

Rayvern is especially useful for:

* 3D artists and modelers
* Game asset management
* Large file collections
* Repetitive renaming workflows
